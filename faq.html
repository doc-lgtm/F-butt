🇬🇧 English — Frequently Asked Questions

What is PowerStudio SCADA?

PowerStudio SCADA is Circutor's energy management software. Its main features are the configuration, communication and monitoring of electrical devices, together with the creation of SCADA screens and reports. It also includes tools such as events, calculated variables, a data export service and a driver management module.

What are the three main components of PowerStudio?

PowerStudio SCADA Editor is the application design tool. PowerStudio SCADA Engine is the communications engine that runs the published application and stores data. PowerStudio SCADA Client is the visualisation module. In addition, PowerStudio SCADA Wave is the web-based administration interface accessible from any browser.

How do I access PowerStudio SCADA Wave?

Open any web browser and enter: https://localhost:8091
The default port is 8091. If changed during installation, use the port specified at that time.

💡 You can create a browser bookmark pointing to https://localhost:8091 to open Wave directly.

Why does the browser show a 'connection is not private' warning?

PowerStudio Wave uses HTTPS with a self-signed certificate. Because this certificate is not issued by a public certificate authority, browsers display a privacy warning on first connection. The connection is safe. To proceed:

  1. Chrome: click Advanced settings, then Access localhost (not secure).
  2. Edge: click Advanced, then Continue to localhost (not secure).
  3. Firefox: click Advanced, then Accept the risk and continue.
📝 The browser will remember this and will not display the warning again on subsequent connections.

What are the minimum system requirements?

  • Processor: Intel Core i5, 10th gen or equivalent. For VMs: minimum 8 cores.
  • RAM: 16 GB
  • Storage: 2 TB
  • OS: Windows Server 2016 or later, or Windows 10 Pro or later.
  • Network: Ethernet adapter required.

How do I activate a license online?

  1. Open https://localhost:8091 and log in as administrator.
  2. Go to Settings → Licenses → Activation online.
  3. Enter your License ID and password, then click Accept.
📝 If the server has restricted internet access, allow https://secure.softwarekey.com through ports 50786 and 443.

How do I activate a license offline?

  1. Go to Settings → Licenses → Offline activation.
  2. Generate the activation request file and save it.
  3. Send the file to iot@circutor.com.
  4. Load the activation file received from Circutor into Wave.

More info: pssdocs.circutor.com/es/modules/license/offline/

How do I migrate from PowerStudio 4.X to Wave?

  1. Back up the Cfg, Editor and Data folders.
  2. Uninstall PowerStudio SCADA 4.X.
  3. Install the new PowerStudio SCADA Wave package.
  4. Open Wave at https://localhost:8091, go to Settings → Licenses and activate.
