Why 2-Step Verification is Required

  • Gmail blocks access from apps that only use username/password for security.
  • Activating 2-Step Verification keeps your account protected even if your password is compromised.
  • Without 2-Step Verification, Gmail will not allow SMTP connections using your normal password, causing authentication errors.

What is an App Password?

  • An App Password is a 16-digit password generated by Google for a specific application.
  • It allows external applications (PowerShell, PowerStudio SCADA, mail clients) to send emails without compromising your main account password.
  • Each App Password is independent, which reduces security risks.

How to Generate an App Password

  1. Activate 2-Step Verification:
    Google Account → Security → 2-Step Verification → Activate
  2. Search for App Passwords:
    In Google Account → Security, use the search bar and type "App passwords".
  3. Open App Passwords and choose an application name (e.g., "PowerStudio SCADA").
  4. Click Create. Google will generate a 16-digit password.
  5. Copy this password and use it in PowerStudio SCADA or your script.
💡 Create a separate App Password for each application. This way you can revoke access to one app without affecting others.
